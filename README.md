# Proyecto HTML: Conquer Academy 

## Descripción

Esta es una página web creada con HTML5. Permite navegar por distintas páginas para informar cuáles cursos tecnolócos están
disponibles y medios de contacto.

---

## ¿Por qué este proyecto?

Este proyecto fue creado como una práctica inicial para aplicar los fundamentos de HTML
aprendidos en el _Máster de Desarrollo Full-Stack de Conquer Blocks_ y cómo estructurar proyectos simples.

---

## Objetivos

- Practicar el uso de HTML5
- Utilizar etiquetas semánticas
- Crear múltiples páginas enlazadas entre sí
- Organizar correctamente la estructura del proyecto

---

## Tecnologías utilizadas
