Curso de Programación FullStack

El curso de Programación FullStack está diseñado para formar desarrolladores capaces de crear aplicaciones web completas, desde el diseño visual hasta la lógica del servidor y la gestión de bases de datos.

A lo largo del curso, los estudiantes aprenderán a desarrollar proyectos reales utilizando tecnologías modernas del mercado.

Objetivos

  • Comprender los fundamentos del desarrollo web
  • Crear interfaces modernas y accesibles
  • Desarrollar aplicaciones web dinámicas
  • Conectar aplicaciones con bases de datos
  • Prepararse para el mercado laboral

Temario

  1. Introducción a la informática y Pseudocódigo
  2. Introducción a Linux y la terminal
  3. Python básico y avanzado
  4. Principios SOLID y acceso a base de datos, Git y Github
  5. Front-end
  6. Back-end
  7. Preparación de entrevistas de trabajo y Metodologías Ágiles/ Scrum
  8. Back-end avanzado
  9. Front-end avanzado

Tecnologías utilizadas

Pseint, Python, HTML5, CSS, JavaScript, ReactJS, TypeScript, SQL, Streamlit, Django, Astro, Angular, VueJS, Java, Node.JS, Rust, Go

Duración y modalidad del curso

Duración estimada: 12 meses.

Modalidad: 100% virtual con clases grabadas.

Requisitos

  • No se requieren conocimientos previos
  • Computadora con acceso a internet
  • Ganas de aprender y practicar
